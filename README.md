# Energy Waste Detection in Smart Homes

## Objective
Detect and predict energy waste patterns using smart home energy and weather data.

## Dataset
Smart Home Dataset with Weather Information (Kaggle)

## Techniques Used
- Data Cleaning & Feature Engineering
- Isolation Forest (Anomaly Detection)
- LSTM Autoencoder
- K-Means Clustering
- Random Forest / XGBoost
- SHAP Explainability

## Key Insights
- Identified abnormal energy consumption patterns
- Detected inefficient appliance usage behavior
- Built predictive model for energy waste

## Tools
Python, Pandas, Scikit-learn, TensorFlow, XGBoost, SHAP
